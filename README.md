# 📂 Automated Backup System (BYOP Project)

## 📌 Project Overview
This project is a simple automated backup system developed using PowerShell. It copies selected folders into timestamped backup directories and maintains logs for tracking operations.

---

## 🎯 Problem Statement
Many users risk losing important files due to system crashes, accidental deletion, or lack of regular backups. This project provides a simple solution to automate backups and maintain records.

---

## ⚙️ Features
- Automatic backup of selected folders
- Timestamp-based backup directories
- Logging of all operations
- Error handling for inaccessible folders
- Easy configuration using `config.ps1`

---

## 🛠️ Technologies Used
- PowerShell
- Windows File System
- Git & GitHub (Version Control)
